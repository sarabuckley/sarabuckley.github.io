About Me

My name is Sara Buckley. I’m a recent graduate of the University of Washington with a B.A. in Integrated Social Sciences. I enjoyed the experience so much, I decided to return to school again, this time to brush on my coding skills. My last gig was at Microsoft, working on the Host Integration Server product. The languages I used then were Visual Basic; SQL Server and COBOL (yup, I even helped fix the Y2K bug!). So far I am loving new languages like Python and Rust.

When I’m not coding, I’m probably hanging out with family, vacuuming up cat hair, playing board games, and enjoying the great outdoors in the Northwest.
